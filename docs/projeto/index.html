Projeto

Novo Projeto​

Para começar, acesse o menu "Projeto" na barra de navegação. Em seguida, clique no botão "Novo projeto" localizado na aba que se abrirá. Essa ação permitirá que você inicie um novo projeto.

Preencha as informações iniciais do projeto de acordo com o formulário e clique no botão "Salvar".

Os campos "Tipo de Inventário" e "Método de Amostragem" possuem opções para seleção. Pelo menos uma opção deve ser informada.

Ao salvar o projeto, ele será aberto automaticamente para preenchimento das demais informações nas abas correspondentes.

Identificação​

Para garantir um registro completo do projeto, é importante preencher adequadamente a aba de Identificação, que contém informações vitais para identificar a propriedade, tipo, método e áreas envolvidas no projeto. Além disso, é fundamental fornecer detalhes específicos sobre cada uma dessas categorias, a fim de facilitar a compreensão e análise do projeto como um todo.

Descrição dos campos:

  • Nome do Projeto: Nome particular de cada projeto
  • Propriedade: Nome da propriedade que o projeto está sendo feito
  • Tipo: Identifica o tipo de projeto que pode ser
  • Método de Inventário: Método utilizado na coleta de amostra
  • Sistema Externo (Controle de Código de Espécies): Sistemas externos onde estão disponíveis cadastros de espécies.
  • Área do projeto (ha): Área total do projeto;
  • Preservação Permanente (ha): Área total de preservações e reservas legal;
  • Área liquida/UPA (ha): Área líquida/UPA de exploração;
  • Cadastro das UT: Cadastro das unidades de trabalho

    Observação: Nesse campo é possível adicionar mais de uma UT. Caso o usuário necessite cadastrar uma ou mais UT, basta adicionar no botão “Adicionar UT” e incluir um novo registro

    • Número UT: Número da unidade de trabalho

    • Área líquida: Área líquida de cada UT (Unidade de Trabalho);

  • Observações: Espaço livre para registro diversos do projeto.

Parâmetros​

O sistema ja possui os parâmetros pré-definidos por padrão.

Descrição dos campos:

  • LE (%): Limite de Erro aceitável na estatística. Valor padrão = 10%;

  • Ciclo de exploração: Tempo (em anos) do ciclo de exploração;

  • IMA: Incremento médio anual;

  • Forma de medida: Forma de coleta dos dados, podendo ser:

    • CAP: Circunferência a altura do peito;
    • DAP: Diâmetro a altura do peito;
    • CAS: Circunferência a altura do solo;

  • Diâmetro de Casca (%): Colocar definição aqui

  • Nº de quadrado/UA: Número de quadrados em uma unidade de amostra para inventário da regeneração da floresta. Aplicado para o manejo florestal.

  • Altura Max. Permitida (m): Altura máxima permitida durante a digitação do inventário. Parâmetro de crítica de digitação.

  • Porta Sementes (%): Percentual de indivíduos a ser selecionado para Porta Sementes. Aplicado para o Plano de Corte.

  • Diâmetro Max. Permitido (m): Diâmetro máximo permitido durante a digitação do inventário. Parâmetro de crítica de digitação.

  • Fatores De Forma: Valor de correção da forma do tronco para determinação do volume. De acordo com a classe de fuste do indivíduo é aplicado o fator, assim sendo CF=1, aplica-se fator de forma 1, CF=2, fator de forma 2, CF=3, fator de forma 3.

  • Método Aleatório

  • Ttab: Valor tabelado extraído da Tabela de Student de "t". Este valor é determinado automaticamente quando LE for igual a 10%.

    Obs.: Este valor deverá ser informado quando o LE for diferente de 10%;

  • Tam. U.A. (ha): Área das unidades de amostras. Valor padrão - Para Amostras Aleatórias e Sistemáticas - 0.25 ha - Para Amostras Conglomerado - 1.00 ha.

    • Tam. U.S. (ha): Área das unidades secundarias, aplicado para Conglomerado. Valor padrão - 0.25 há;

  • Classes diâmétricas: Determinação das classes dos diâmetros (metros), para classificação de indivíduos nos relatórios;

  • Aproveitamento: Determinação do diâmetro explorável e remanescente explorável da espécie segundo sua utilização.

Ao término da inclusão dos dados na aba parâmetro. Clicar na aba “Inventário”

Inventário​

Na aba Inventário, é necessário preencher com os dados coletados durante as atividades de campo. Esses dados incluem informações como a identificação da parcela, número do indivíduo, medição realizada, classe de fuste quando aplicável, e outras informações relevantes conforme necessário, como a identificação das árvores. É importante garantir que todos os dados sejam registrados de forma precisa e completa, incluindo a identificação das árvores, para uma análise adequada posteriormente.

  • Parcelas: Dependendo do tipo de projeto, considere o termo Parcela como Amostra ou Picada.

    • Número: Número de identificação da Parcela (Amostra ou Picada);
    • Talhão: Número de identificação do talhão de exploração;
    • Coordenadas Planas-Início X e Y, Fim X e Y: Representam as coordenadas de latitude e longitude do início e fim da picada. Estas coordenadas se aplicam apenas para o Plano de Corte, para plotagem dos indivíduos. Veja os exemplos:

  • Medição: Considerando que para o inventário contínuo, faz-se n medições na mesma parcela por um período de tempo, este quadro identifica estas medições. Para os demais projetos, haverá somente uma medição.

    • Número: Número da n-ésima medição das parcelas;
    • Dt. Medição: Data da medição;
    • Atual: Horário de início e fim da medição para determinação de tempo total de medição;

  • Indivíduos: Relação dos indivíduos relativa à Parcela e Medição. Selecionando nova Parcela ou Medição, serão selecionados seus respectivos indivíduos.

    • Ind (Individuo): Número sequencial do indivíduo;
    • Cond (Condição): Condição do indivíduo, sendo: 1-Existente, 2-Extraída, 3-Aproveitada, 4-Morta, 5-Danificada;
    • Código: Código da espécie listado no Cadastro de Espécies;
    • Alt (Altura): Altura do indivíduo, em metros;
    • Med (Medida do Individuo): Determinada pelo parâmetro forma de medida, podendo ser CAP (Circunferência na altura do peito), DAP (Diâmetro na altura do peito) ou CAS (Circunferência na altura do solo);
    • CF (Classe de Fuste): sendo: 1-Reto, 2-Semi-Tortuoso, 3-Tortuoso. Para cálculo do volume do indivíduo aplica-se o fator de forma 1, 2 ou 3 respectivamente;
    • QD (Quadrado na amostra): Número do Quadrado na amostra (parcela) dos indivíduos inventariados da regeneração.
    • SF (Sanidade do Fuste): Sanidade do Fuste, sendo, 1-Livre de doenças, 2-Mediamente Atacada, 3-Altamente Atacada;
    • Lado: Lado onde a espécie se encontra em relação ao início da picada, sendo, 1 - Esquerdo, 2 - Direito e 3 - Centro;
    • Dist.: Distância lateral que o indivíduo se encontra da picada, em metros;
    • Comp.: Comprimento na picada, em metros;
    • Apl (Aplicação): Denominação a que o indivíduo se aplica, sendo: 1-Corte, 2-Porta sementes, 3 - Abatida, 4 - Remanescente, 5 - Corte proibido, 6 - Proibido de Corte, 7-Rara;
    • QC (Qualidade da Copa): sendo, 1 - Adequada, 2 - Intermediaria, 3 - Inadequada;
    • CP (Presença de Cipós): sendo, 1 - Presente, 2 - Ausente;
    • PC (Posição da Copa): , sendo, 1 - Dominante, 2 – Semi dominante, 3 - Dominada;
    • DM (Distância da Matriz): Distancia em metros, sendo,1 - DM<=5m,2-5m>DM<=10m ou 3-DM>10m
    • Vol (Volume): Volume com casca do indivíduo, já aplicado o respectivo fator de forma.

Estatística​

O módulo de Estatística é uma ferramenta essencial nos projetos de Plano de Exploração, Fitossociológico, Manejo e Inventário Contínuo. Ele desempenha um papel importante no processamento de cálculos estatísticos para verificar se a amostragem realizada foi aplicada corretamente.

Ao utilizar esse módulo, é recomendado clicar no botão "Atualiza" após digitar todas as amostras. Isso permitirá que os cálculos estatísticos sejam realizados com base nos dados coletados. Caso seja necessário fazer alterações na amostragem, é fundamental executar novamente a função de atualização para garantir a precisão dos resultados.

As fórmulas aplicadas nos cálculos estatísticos variam de acordo com o método de amostragem utilizado. Essas fórmulas são apresentadas de forma detalhada no Relatório de Estatística, que pode ser acessado através do Gerenciador de Impressão.

Portanto, o módulo de Estatística oferece uma análise minuciosa do inventário, verificando se a amostragem foi bem aplicada e fornecendo informações estatísticas relevantes para o projeto.

Classificação das espécies​

O módulo de Classificação das Espécies permite uma personalização das espécies encontradas no inventário. Um exemplo de personalização seria alterar o nome vulgar de uma espécie para expressar o nome conhecido na região ou mesmo mudar o uso da espécie para atender a necessidade do projeto sendo processado, sem a necessidade de alteração no Cadastro de Espécies.

Clique no botão Atualizar para atualização da relação das espécies encontradas. A cada novo clique são excluídas as espécies e incluídas novamente, ou seja, a personalização das espécies deve ser feita somente após todas as correções do inventário terem sido processadas.

Para manutenção das espécies, utilize a barra de botões do formulário, cuja funções são as mesmas do Módulo de Inventário.
